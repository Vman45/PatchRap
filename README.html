Please note that peripherials directly interacting with logic signals (ie. limit switches) are expected to use 5V power. For use with 3.3V logic, a 3.3V regulator may be provided onboard or by an inline RJ45 connectorized PCB.

For new designs, limit switch common terminals should be used as outputs. A three-terminal limit switch NO pin can be wired to Ethernet/T568A pin2 (sGND). The entire NC line (pin4) can then be wired to 3.3V, with the currently assigned NO line (pin5) wired to common output. In summary, pin2 becomes NO/GND, pin4 HIGH, pin5 SIGNAL.
